# GDELT / CAMEO codebooks

Upstream documentation bundled here for offline reference. The codebooks define the semantics of every GDELT field and are the canonical source for the lookup tables in [`data/lookups/`](../../../data/lookups/).

| File | Description |
|---|---|
| [**GDELT-Event_Codebook-V2.0.pdf**](http://data.gdeltproject.org/documentation/GDELT-Event_Codebook-V2.0.pdf) | Official GDELT 2.0 Event database codebook with field-level definitions for the `events` and `mentions` streams. |
| [**CAMEO.Manual.1.1b3.pdf**](http://data.gdeltproject.org/documentation/CAMEO.Manual.1.1b3.pdf) | CAMEO Event and Actor Codebook with full hierarchical taxonomy used by GDELT's `EventCode`, `EventBaseCode`, `EventRootCode`, `QuadClass` and the CAMEO actor code system. |
